案 A の狙い

シネマティック・ダーク — コンテンツを主役に、UI は闇に溶ける

黒〜濃紺の没入型。大きなヒーローと繊細なグロー・金の差し色でサムネイルを最大限に引き立て、夜のリビングで「今夜の一本」をじっくり選ぶ体験に寄せる。落ち着いた上質感(Netflix / U-NEXT 系)。

没入上質夜のゴールデンタイムサムネが主役

案 B の狙い

クリーン・ライト — 白と余白で「すっと読める」軽快さを最優先

白基調・広い余白・角丸。カードの境界は薄いリングだけに抑え、朝昼夜いつ開いても疲れない高可読の器にする。爽やかで万人向け、家族の共用端末にも馴染む(TVer / NHK+ 系)。

軽快高可読万人向け清潔感

案 C の狙い

ボールド・ビビッド — ランキングを主役に、テレビをお祭りにする

高コントラスト・高彩度・大きなタイポ。ランクバッジをステッカーのように立たせ、「いま盛り上がっている」熱量で若年層の指を止める。太い輪郭とハードシャドウでエンタメの勢いを画面に出す(ABEMA / 若年エンタメ系)。

熱量ランキング主役若年向け高彩度
